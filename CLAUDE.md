# Claude Instructions

## Changelog

После каждого фикса, фичи или любого значимого изменения — обновляй `CHANGELOG.md`.

Формат записи:
```
## [версия] - YYYY-MM-DD

### Added / Fixed / Changed
- Краткое описание на языке пользователя
```

Версию берёшь из `package.json`. Дату — текущую.

Пользователь также может попросить написать changelog явно ("создай changelog", "обнови changelog") — в этом случае смотришь на `git log`, выводишь все изменения с момента последней записи в `CHANGELOG.md` и обновляешь файл.
