# Changelog

## [1.4.0] - 2026-03-05

### Added
- Разрешение веса продуктов по API при нестандартных единицах (порц., ломт. и т.д.); вывод в граммах
- В JSON `meals --format json`: поле `total` за день и в каждом item поле `amount_display` (г/мл по-русски)
- Команда `day`: разметка Telegram (*жирный*, _курсив_), дата на русском, КБЖУ по каждому приёму пищи

## [1.2.0] - 2026-03-05

### Added
- В команде `meals` выводится КБЖУ по каждому приёму пищи (калории, белки, углеводы, жиры). В таблице/CSV — колонки по приёму, в JSON — структура `meals[приём].items` и `meals[приём].nutrients`

## [1.0.2] - 2025-03-05

### Added
- Команда `day` — сводка по конкретному дню: калории/макросы (с % от цели), шаги, вес (и цель по весу), приёмы пищи
- Кэш токена в `yazio-token.json` (текущая директория), опция `--no-cache-token` для отключения

## [1.0.1] - 2026-03-02

### Fixed
- Экспресс ввод (simple_products) теперь отображается в команде `meals`

## [1.0.0] - 2026-02-24

### Added
- Команды: `summary`, `meals`, `water`, `weight`, `exercises`, `goals`
- Форматы вывода: `table`, `json`, `csv`
- GitHub Actions для автоматической сборки релизов
